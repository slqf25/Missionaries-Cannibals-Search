# Missionaries-Cannibals-Search

A Python implementation of multiple AI search algorithms to solve the classic **Missionaries & Cannibals** river crossing problem.  
Includes performance comparison across test cases and console-based step-by-step visualization.

##  Project Overview
The Missionaries & Cannibals puzzle is a well-known AI search problem involving a boat that can carry at most two people at a time, with the rule that missionaries must never be outnumbered by cannibals on either bank.  
This project demonstrates how different search algorithms explore the problem space and compares their **completeness**, **optimality**, **time complexity**, and **space complexity**.

##  Features
- Implemented algorithms:
  - **Breadth-First Search (BFS)**
  - **Depth-First Search (DFS)**
  - **A\* Search**
  - **Greedy Best-First Search**
  - **Simulated Annealing (SA)**
- Multiple test cases for varied difficulty
- Console-based animation of moves
- Fixed-speed visualization for consistent demonstrations
- Detailed performance metrics:
  - Path length
  - Execution time
  - Nodes expanded
  - Maximum frontier size
- SA supports multiple runs to improve success rate
